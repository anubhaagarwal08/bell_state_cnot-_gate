# Bell State using Qiskit

This repository demonstrates the implementation of Bell States using IBM Qiskit.

## Files

- `bell_state_simulator.py` – Executes the Bell State circuit using AerSimulator.
- `bell_stategit .py` – Executes the Bell State circuit on IBM Quantum hardware using IBM Runtime.
- `cnot_gate.py` – Demonstrates the operation of the Controlled-NOT (CX) gate.

## Requirements

- Python
- Qiskit
- Qiskit Aer
- Qiskit IBM Runtime
